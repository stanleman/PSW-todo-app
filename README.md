Run ```pub get``` to get necessary dependencies.

Create a .env file to get Firebase working.
```
# Universal
STORAGE_BUCKET=***
PROJECT_ID=***
MESSENGER_ID=***

# Web / Windows
WEB_WIN_API_KEY=***
WEB_APP_ID=***
AUTH_DOMAIN=***

# Android
ANDROID_API_KEY=***
ANDROID_APP_ID=***

# iOS / macOS
IOS_MAC_API_KEY=***
IOS_MAC_APP_ID=***
IOS_MAC_BUNDLE_ID=***

```
Replace *** with your Firebase information.

Run main.dart to start the app.
